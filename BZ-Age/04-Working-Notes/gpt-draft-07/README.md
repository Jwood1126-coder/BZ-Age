## Editorial Identity

This is a radical compression and orientation pass. It deliberately tests whether each part can be told with:

- fewer proper nouns;
- clearer region roles;
- stronger "what changed?" spines;
- more ordinary-body pressure;
- less catalogue energy;
- stronger Aegean/Mycenaean callback value;
- clearer collapse mechanics;
- a finale that moves from memory to recovery to the privileged modern vantage.

Because it is compressed, it sacrifices some of draft-05's evidentiary richness. Treat it as a structural/audio audition: useful for comparison, not an adoption-ready replacement.

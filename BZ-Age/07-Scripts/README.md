## Drafting Notes

- Keep script work in this directory.
- Do not use script-drafting metadata as a reason to edit `CLAUDE.md`.
- Keep source-check flags until the relevant source-map entry exists.
- When a claim is source-hardened, replace the flag with a source key rather than simply deleting it.
- Drafts should remain audio-first: short paragraphs, spoken transitions, clear callbacks, and explicit uncertainty where needed.

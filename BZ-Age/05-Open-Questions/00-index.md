# Open Questions — Index

The research questions, decisions to make, and open debates that the project still has to chase down. This is the issue tracker for a research project.

Each question gets logged here with the part it relates to (or "cross-cutting"), when it was raised, and what would resolve it.

## Convention

- Use a heading for each open question.
- Tag it with the part it relates to.
- State what kind of resolution would close it (a source consulted, a decision made, a beat developed).
- When a question is resolved, **don't delete it** — strike through the question and add a brief note about how it was resolved, with the date.

# Source Map — Index

The working bibliography for the series. Organized by part. Each part will contain primary sources, modern scholarly works, archaeological sites, accessible entry points, and where the field has moved recently.

**Status:** Skeleton. Full population is the next major Phase One deliverable.

## Working convention

Each source gets:

- **One-paragraph summary** of what it is and what it covers
- **Drill-down recommendations** — which letters, tablets, chapters, sections are gold for narrative purposes
- **Strengths and weaknesses** — what it's good for, what it's dated or weak on
- **Disagreements** — where this source clashes with others
- **Ground-level flag** — whether it's load-bearing for ground-level material specifically

## Recommended start order

Start with Part 1 (chronological logic + early-period sources are some of the trickiest, so getting them right early provides a foundation). Part 5 is defensible as an alternative (richest material, will teach us most about how to handle the dense documentary periods).
